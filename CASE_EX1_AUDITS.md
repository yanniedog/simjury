## 1. How to read this document

Each case has an **access status**:

| Status | Meaning |
|--------|---------|
| **CONFIRMED** | Classes (b)–(e) identified; authoring may proceed after tabulation |
| **PARTIAL** | Primary NBT or class (b) missing; substitute path documented |
| **PD VERIFY** | Sources exist but publication year needs operator PD sign-off |
| **HARNESS BLOCKED** | Fails I-4 or other exclusion — do not author |
| **SOURCE BLOCKED** | Harness-eligible but required documents not locatable |

**Source classes** (v3 §8.2): **(a)** OBO text [excluded]; **(b)** Sessions Papers images; **(c)** official reports; **(d)** newspapers; **(e)** pardon/parliamentary aftermath.

**Transcription rule:** Cite class **(b)** or **(c)** locators in JSON. NBT is an auxiliary cross-check only unless no class (b) exists — then NBT becomes primary with operator PD confirmation.
