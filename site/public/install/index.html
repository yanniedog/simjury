Separate Android pilot

Your offline courtroom.

Install once, then play the included pilot case without a connection. There is no account or tracking; optional update checks are the app's only network use.

Download SimJury (.zip)

Why a zip? Some mobile browsers remove bare APK downloads. The zip keeps the installer intact until you extract it. The files come directly from SimJury's public GitHub release.

Install in six steps

  1. Tap Download SimJury (.zip) above.
  2. Open Files → Downloads, tap the zip, and Extract.
  3. Tap the extracted app-preview.apk to install.
  4. If prompted, allow installs: Settings → Install unknown apps → Files → Allow. You can turn this permission off again afterwards.
  5. If Play Protect blocks it: More details → Install anyway.
  6. If you see a signature mismatch, uninstall any older SimJury first, then reinstall.
